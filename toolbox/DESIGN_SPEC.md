## 1. 整体视觉风格

| 属性 | 值 |
|------|----|
| 背景色 | `#000000`（纯黑） |
| 主色 | `#00FF41`（磷光绿，仿老式 CRT 终端） |
| 强调色（警告/录音） | `#FF3B30`（红） |
| 非活跃元素 | `#1A4A1A`（暗绿，dimmed） |
| 字体风格 | 等宽像素字体（内置 font0/font2，或自定义 5×7 bitmap font） |
| 时间数字 | 大号等宽字，底部左侧，视觉重心 |
| 整体气质 | 复古磁带机 + 医疗心电图 + 终端 CLI 的混搭美感 |

## 2. 全局布局

```
┌────────────────────────────────┬───┐
│                                │ 列 │  ← 右侧标签栏，宽 14px
│        主内容区                 │ 表 │
│        (226×135)               ├───┤
│                                │ 回 │
│                                │ 放 │
│                                ├───┤
│                                │ 录 │
└────────────────────────────────┤ 制 │
                                 └───┘
```

### 右侧标签栏（Tab Bar）— 宽 14px，贴右边

- 三个等高区域各占 45px 高
- **非活跃**：细边框矩形 `□`，暗绿色，字体暗绿
- **活跃**：实心绿色矩形 `■`，向左凸出 3px（贴边弹出效果），字纯白或亮绿
- 文字竖排：`列表` / `回放` / `录制`，字号 8px

### 电量（右上角，叠加在主内容区右上）

- 位置：主内容区右上角，坐标约 `(170, 2)`
- 格式：`XX%`，字号 8px，亮绿色
- 电量 ≤ 20% 时变红色并闪烁

## 3. 录音界面

### 3.1 界面布局

```
┌──────────────────────────────────────┐
│ ● REC              65%               │  ← 顶栏 14px 高
├──────────────────────────────────────┤
│                                      │
│   [实时波形图，滚动显示]               │  ← 波形区 80px 高
│                                      │
├──────────────────────────────────────┤
│ 03:34                                │  ← 底栏 26px 高，左下时间
└──────────────────────────────────────┘
```

### 3.2 各元素规格

#### 顶栏
- **红点**：半径 5px 圆，`#FF3B30`，位于 `(8, 7)`，**1Hz 闪烁**（录制中）/ 录制停止后熄灭
- **"REC" 文字**：紧跟红点右侧，亮绿色，等宽字体 font2（10px），录制中显示
- **电量**：右上角，见全局布局

#### 波形区（实时录音波形）
- 区域：`(0, 14)` → `(226, 94)`，高度 80px，黑色背景
- 波形：**从右向左滚动**（新数据从右侧进入，旧数据向左推出）
- 颜色：`#00FF41`
- 振幅：以中线（y=54）为基准，最大上下各 36px
- 采样：每帧从 mic buffer 取平均振幅，绘制 1px 宽竖线段
- 中线：可选绘制 1px 细横线 `#1A4A1A`（极暗，不抢眼）

#### 时间计数器
- 位置：底部左侧 `(4, 110)`
- 格式：`MM:SS`
- 字号：**最大可用**（约 font4 / font6，高度 18–22px）
- 颜色：`#00FF41`
- 从 `00:00` 开始向上计时，精确到秒

#### 右侧标签栏
- **录制** 标签：活跃（实心绿，向左弹出 3px）
- 其余两个：非活跃（暗绿空心）

### 3.3 状态机

```
进入录音界面（不录制）
    ↓ 按空格
开始录制 → 红点闪烁，波形滚动，计时器启动
    ↓ 再按空格
结束录制 → 红点熄灭，波形停止，文件写入 SD
    ↓
    ↓ 录制中按其他键（N/回车/方向键等）
结束录制 + 跳转到对应界面
```

## 4. 播放界面

### 4.1 界面布局

```
┌──────────────────────────────────────┐
│                           65%        │  ← 顶栏（仅电量，无标题）
├──────────────────────────────────────┤
│                                      │
│   [波形进度条 / 静态波形图]            │  ← 波形+进度区 80px 高
│   ──────────┼──────────────────────  │  ← 中轴线
│                                      │
├──────────────────────────────────────┤
│ 03:34                                │  ← 底栏左下时间
└──────────────────────────────────────┘
```

### 4.2 波形进度条

这是播放界面最核心的功能组件，将传统进度条升级为可视化波形进度条。

#### 实现原理
- 打开 WAV 文件后，**均匀采样 `WAVE_BARS` 个点**（建议 56 个，对应波形区宽度）
- 每个采样点取该区间的 **绝对值平均振幅**，归一化到 0–1
- 绘制为**以中轴线为基准、上下对称的竖线**（类似 ECG 风格）

#### 视觉规格
- 已播放部分：亮绿 `#00FF41`，全亮度
- 未播放部分：暗绿 `#1A4A1A`（dimmed 25%），或灰绿 `#004400`
- 播放头（playhead）：1px 宽亮白竖线 `#FFFFFF`，随进度移动
- 中轴线：1px `#003300`，贯穿全宽

#### 刷新策略
- 波形数组**只在文件打开时加载一次**（不每帧重算）
- 播放头位置每 200ms 更新一次（降低 SD 读写压力）

### 4.3 时间显示

- 格式：`MM:SS`（当前播放位置）
- 位置、字号与录音界面相同

### 4.4 磁带轮动画（可选，原稿提及）

- 在波形区右侧或底部居中绘制两个小圆圈，模拟磁带卷轴
- 播放中顺时针慢速旋转（每秒约 1/4 圈）
- 快进时加速旋转
- 停止时静止
- 实现方案：绘制带缺口的圆弧（2个弧段），每帧旋转偏移角度

### 4.5 右侧标签栏
- **回放** 标签：活跃
- 其余：非活跃

### 4.6 按键行为

| 按键 | 行为 |
|------|------|
| 回车 | 播放/暂停切换 |
| 长按右方向键 | 快进（2×/4× 速，松开恢复）|
| 长按左方向键 | 倒带（反向 2× 速，或按段回跳）|
| 短按右方向键 | 进度 +5 秒（可选）|
| 短按左方向键 | 进度 -5 秒（可选）|
| 空格 | 进入录音界面 |
| N | 进入列表界面 |

#### 快进/倒带视觉反馈
- 快进时：磁带轮加速旋转，播放头加速移动
- 波形区顶部短暂显示 `▶▶` 或 `◀◀` 图标（500ms 后消失）

## 5. 列表界面

> 原稿标注「你来完善」，以下为补充设计建议，可接受修改。

### 5.1 界面布局

```
┌──────────────────────────────────────┐
│ 录音列表                  65%        │  ← 顶栏，标题 + 电量
├──────────────────────────────────────┤
│ > [001] 2024-01-15  03:34  ████░░░  │  ← 选中行，高亮
│   [002] 2024-01-15  01:22  ██░░░░░  │
│   [003] 2024-01-16  00:45  █░░░░░░  │
│   [004] 2024-01-16  05:12  ███████  │
│   [005] 2024-01-17  02:08  █████░░  │  ← 最多显示 5 行
├──────────────────────────────────────┤
│ ↑↓选择  回车播放  DEL删除            │  ← 底部操作提示栏
└──────────────────────────────────────┘
```

### 5.2 列表行设计

每行高度：**22px**，可显示约 5 行。

| 元素 | 内容 | 宽度 | 颜色 |
|------|------|------|------|
| 序号 | `[001]` | 32px | 暗绿 |
| 文件名/日期 | `rec_001` 或从文件名解析 | 80px | 亮绿 |
| 时长 | `03:34` | 36px | 亮绿 |
| 迷你波形预览条 | 7 格小方块 ████░░░ | 48px | 亮绿/暗绿 |

- **选中行**：整行填充暗绿背景 `#003300`，文字高亮 `#FFFFFF` 或 `#00FF41`
- **选中游标**：`>` 符号在行首

### 5.3 滚动行为
- 列表超过 5 项时滚动显示
- 选中项始终保持在第 2–4 行（居中对齐，避免贴顶/贴底）
- 右侧可加 1px 宽滚动条（亮绿色细线，按比例）

### 5.4 删除流程

```
列表界面（正常模式）
    ↓ 按 DEL
弹出确认对话框（覆盖在列表上）：
  ┌─────────────────────┐
  │  删除 [rec_001] ?   │
  │  回车确认  其他键取消 │
  └─────────────────────┘
    ↓ 回车
文件删除，列表刷新，光标移到上一项
    ↓ 其他键
取消，回到列表
```

- 对话框样式：居中，有 2px 绿色边框，黑色背景，确认字红色

### 5.5 长按 DEL（任何界面可用）

从任何界面长按 `DEL` 键（约 500ms）→ 直接跳转到列表界面。

### 5.6 右侧标签栏
- **列表** 标签：活跃
- 其余：非活跃

## 6. 按键交互总表

| 按键 | 录音界面 | 播放界面 | 列表界面 |
|------|----------|----------|----------|
| **空格** | 开始/停止录制 | 进入录音界面（不开始录制）| 进入录音界面 |
| **回车** | — | 播放/暂停切换 | 播放选中录音 |
| **N** | 进入列表 | 进入列表 | —（无效）|
| **↑** | — | — | 上移选中 |
| **↓** | — | — | 下移选中 |
| **←**（短按） | — | 倒退 5s（可选）| — |
| **→**（短按） | — | 前进 5s（可选）| — |
| **←**（长按） | — | 倒带（加速）| — |

## 7. 动画与特效说明

### 7.1 红点闪烁（录音中）
- 频率：**1Hz**（亮 500ms → 灭 500ms）
- 实现：`millis()` 取余，奇偶切换绘制/不绘制

### 7.2 录音波形滚动
- 每帧向左平移 2px，新数据从右侧补入
- 使用帧缓冲或逐列移位绘制（不 `fillRect` 清屏，用 `scroll` 或逐像素移位）

### 7.3 磁带轮旋转
- 绘制：两个圆（左轮/右轮），每个圆内画 3 条辐射线（120° 间隔）
- 状态：停止/播放/快进，分别对应不同角速度
- 实现：存储当前角度变量 `float reelAngle`，每帧累加 `Δθ`，绘制旋转后的辐射线

### 7.4 标签栏弹出效果
- 活跃标签**向左凸出 3px**（宽度从 14px 扩到 17px）
- 切换界面时，旧标签缩回、新标签弹出，做 **2 帧渐变**（0→1→2→3px 凸出量）

### 7.5 快进/倒带图标
- 位置：波形区顶部中央
- 内容：`>>` 或 `<<`，字号 font2
- 出现时机：开始长按时显示，松键后 500ms 消失

### 7.6 界面切换
- 不做复杂动画（资源有限）
- 切换时执行一次 `fillScreen(BLACK)` 全清屏即可
- 可选：做 2 帧的扫描线效果（从上到下逐行清除）

## 8. 优化建议（超出原稿）

以下为在原稿基础上的设计优化，实现时可选采纳：

### 8.1 文件名显示优化
- 列表行中如果文件名过长，做**横向滚动**（选中行自动 marquee）
- 或改用「recNNN」统一序号显示，右侧标注文件时长

### 8.2 播放界面增加文件标题行
- 波形区顶部叠加显示当前文件名（小字，暗绿色），不影响波形

### 8.3 电量动态图标
- 在百分比左侧加 3×6 px 的电池图标（3 格像素块），按电量填充格数
- 低电量时整个电量显示区红色闪烁

### 8.4 录音界面增加文件序号预告
- 顶栏 "REC" 右侧显示将要保存的文件号，如 `REC #042`
- 帮助用户了解录音存在哪里

### 8.5 列表界面迷你波形预览
- 每行右侧 7 格小方块代表该录音的振幅分布（粗略缩略图）
- 打开列表时后台异步读取（或复用已缓存的波形数据）

### 8.6 静音检测（可选高级功能）
- 录音超过 3 秒静音自动暂停，避免录到大段空白
- 波形区静音段显示为虚线而非实线
