# Cardputer 按键与快捷键备忘

更新时间：2026-05-28

## 全局按键定义

- `Esc`：物理左上角 `` ` `` 键。
- `上传键`：`\` 或 `|`。
- `上 / 下 / 左 / 右`：`;` / `.` / `,` / `/`。
- `音量 + / -`：`=` 或 `+` / `-` 或 `_`。
- `亮度 + / -`：`]` 或 `}` / `[` 或 `{`。
- `Tab`：当前作为组合修饰键使用；按住 `Tab` 时音量和亮度快捷键不会响应，避免误触。
- `Ctrl`：目前不再承担旧组合键功能，预留给后续新用途。

## 息屏 / 启动入口

- `Space`：进入录音。
- `Enter`：进入录音列表。
- `F`：进入番茄钟。
- `C`：进入聊天。
- `Go / BtnA`：进入应用列表。

## 录音中

- `Space`：暂停 / 继续录音。
- `Enter`：结束录音，保存并回到录音列表。
- `Esc`：保存录音并息屏。
- `长按 Del 1.2 秒`：取消并删除本条录音。
- `上传键`：结束录音，保存后加入上传队列并回列表。
- `Tab + 上传键`：取消当前上传队列 / 上传动作。
- `Tab` 单按：录音页熄屏。
- `Alt`：结束录音后进入降噪流程。
- `音量 + / -`：调节播放/监听音量。
- `亮度 + / -`：调节屏幕亮度。
- `W / S`：调节录音增益。

## 录音列表

- `左 / 右`：切换 `NOR / KEY / IMP` 列表。
- `上 / 下`：移动选中录音。
- `Enter`：播放当前选中的录音。
- `Space`：开始新录音。
- `Esc`：退出列表并息屏。
- `长按 Del 1.2 秒`：删除当前选中录音。
- `Tab + 字母/数字`：把当前选中录音绑定到对应快捷键。
- `字母/数字快捷键`：播放已绑定的录音。
- `Tab + Enter`：把当前选中录音标记为重要。
- `Tab + Del`：删除未标记录音。
- `Alt`：对当前选中录音降噪。
- `上传键`：
  - 未上传 / 失败：加入上传队列。
  - `SN`：立即刷新服务器状态。
  - `OK`：只提示 `OK`，不重复上传。
- `Tab + 上传键`：取消上传队列 / 上传动作。

## 回放页

- `Enter`：暂停 / 继续播放。
- `Esc`：息屏。
- `Space`：去录音。
- `上 / 下`：上一条 / 下一条。
- `左 / 右`：快退 / 快进。
- `字母/数字快捷键`：切换播放对应绑定录音。
- `Del`：
  - 普通录音：长按删除当前录音。
  - 绑定快捷键的录音：停止播放并返回列表。
- `音量 + / -`：调节全局播放音量。
- `Alt + 音量 + / -`：只调节当前录音的相对音量，不影响其他录音。
- `亮度 + / -`：调节屏幕亮度。
- `上传键`：当前只显示 `LIST` 提示，不触发上传。
- `Tab + 上传键`：当前只显示 `LIST` 提示，不触发上传。

## 聊天

- `Enter`：开始说话 / 发送 / 进入对应聊天动作，具体取决于当前聊天页面。
- `Space`：录音入口或按住说话，取决于当前聊天页面。
- `Esc`：返回 / 取消。
- `C`：聊天页内的快捷动作键。
- `亮度 + / -`：聊天页用于切换主题色。

## 番茄钟

- `Enter`：开始 / 暂停。
- `Del`：重置。
- `Esc`：返回。
- `Space`：进入录音。
