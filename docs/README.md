# 文档地图

这个文件夹集中放项目说明、交接和历史参考。根目录的 `项目备忘.md` 仍是开新窗口时最先看的总入口。

## 固件

- `firmware/HANDOVER.md`：Cardputer 固件交接和当前状态。
- `firmware/COMPILE_FLASH.md`：编译、烧录、Arduino CLI 和核心版本说明。
- `firmware/DESIGN_SPEC.md`：录音工具 UI 与交互规格。

## 服务器

- `server/README.md`：cloud voice server 的基本说明和接口。
- `server/DEPLOYMENT.md`：阿里云 ECS、nginx、systemd 部署说明。
- `server/SERVER_MEMO.md`：服务器运行备忘、数据文件、转写/flomo 逻辑、后续音频处理方向。

## 历史参考

- `archive/CLOUD_TRANSCRIBE_FLOMO_PLAN.html`：云端转写与 flomo 同步早期方案。
- `archive/MULTI_APP_ARCHITECTURE_ADVICE.html`：多应用架构建议。
- `archive/HANDOVER_claude.html`：旧版 HTML 交接文档。
- `archive/README_20260524_stable_sleep_playback_fade.md`：稳定版备份说明。
- `archive/font_preview.html`：字体预览实验页面。

## 工具页

`toolbox/NET_CONFIG_HELPER.html` 仍保留在 `toolbox/`，因为它是实际生成 `net.txt` 的工具页，不只是说明文档。
