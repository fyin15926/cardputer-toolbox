# Cardputer-Adv 工具箱 项目备忘

> 最后更新: 2026-05-22。给自己和 AI 看的速查手册。

## 一、这是什么
给 M5Stack **Cardputer-Adv** 写的一个"工具箱"固件(Arduino/C++)。
- 风格: 纯黑底 + 黑客帝国荧光绿 + 中文界面。
- 主页: `[空格] 录音`  `[回车] 播放最近`。
- 录音机: 实时绿色声波(示波器), 录音中 **W 调大 / S 调小** 音量, **其他键停**。
- 录完结果页: `[回车] 播放`  `[N] 降噪`(FFT 谱减法)  其他键返回。
- 录音存成 16kHz 单声道 WAV, 放在 SD 卡 `/REC/REC_xxxx.wav`。

## 三、怎么编译 + 烧录(核心流程)
1. 用**能传数据的 USB-C 线**把机器连到电脑(劣质/充电线会"无法识别 USB", 换线!)。
2. 电源开关 ON。
3. 在 PowerShell 里跑(端口一般是 COM3, 不确定先用 `arduino-cli board list` 查):
```
$cli="C:\cardputer\tools\arduino-cli\arduino-cli.exe"
& $cli compile --fqbn esp32:esp32:m5stack_cardputer C:\cardputer\sketches\toolbox
& $cli upload  --fqbn esp32:esp32:m5stack_cardputer -p COM3 C:\cardputer\sketches\toolbox
```
- **开发板型号(FQBN)**: `esp32:esp32:m5stack_cardputer`(库会自动识别 Adv)。
- 如果烧不进/认不到: 进**下载模式** = 电源 OFF → 按住 G0 → 通电 → 松开 G0, 再烧。
- 看程序崩溃日志: 用串口工具连 COM3 @115200, 会打印 `Guru Meditation` 等。

## 四、关键的坑和决定(很重要, 别重复踩)
1. **核心必须用 esp32 3.1.3, 不能用 3.3.x!**
   - 3.3.8(最新)上**麦克风录不了音**(录到静音/平线), 这是新核心的 I2S 录音回归 bug。
   - 3.1.3(IDF 5.3)麦克风正常。降级方法见 `C:\cardputer\prefetch313.ps1`。
2. **键盘库被改成"轮询"了**(去掉了 GPIO 中断)。
   - 文件: `...\M5Cardputer\...\KeyboardReader\TCA8418.cpp`。
   - 原因: 原版用 GPIO 中断, 在 3.1.3 上开机会撞爆系统 ipc1 任务栈 → 不停重启/闪屏。
   - ⚠️ **如果重装 M5Cardputer 库, 这个改动会丢, 要重新改**(把 begin() 里的 attachInterrupt 删掉、update() 改成 `while(available()) getEvent...` 轮询)。
3. **开机会"预热"麦克风**(空跑一次录音会话 + 暖一下 SD), 否则**冷启动后第一次录音音量小、断续**。
4. **国内下载慢**: esp32 核心用 Jihulab 镜像 + 从 `dl.espressif.cn` 用 curl 高速预取到缓存, 再让 arduino-cli 从缓存装(arduino-cli 自带下载器对国内 CDN 很慢)。
5. **这颗芯片(ESP32-S3FN8)没有 PSRAM** → 跑不了 Espressif ESP-SR 专业降噪。所以只能用自写的轻量谱减法降噪。
6. 喇叭是 1W 小喇叭, 本身就小声; **要大声清晰用 3.5mm 耳机**。

## 五、可调参数(都在 toolbox.ino 里)
| 参数 | 位置 | 作用 |
|---|---|---|
| `recGain`(默认 50) | 文件顶部 | 录音音量(也可录音时 W/S 现场调) |
| 低通系数 `160` | recorderApp 的 lpf 那行 | 越小越闷、压高频底噪越多 |
| `HEAD_SKIP`(13) | recorderApp | 掐头时长(去开始键声) |
| `tailTrim`(REC_RATE/4) | 收尾处 | 掐尾时长(去结束键声) |
| 软限幅 `tanhf` | 每采样处理 | 防大声"哒哒"爆裂 |
| 降噪 `ALPHA`(3.0)/`BETA`(0.05) | noiseReduce() | 谱减强度/谱底; ALPHA 越大降噪越狠但易有"水声" |
| 喇叭 DAC `0x32` | playWav() | 回放音量微调 |
